# Global Meat Consumption Dashboard (1990-2028) **From the Kaggle Dataset**

Interactive Power BI analysis of meat consumption trends by country & type.

![Dashboard Preview] (Screenshot 2026-03-20 142626.png)  

**Answers These 5 Questions:**

| Question | Visual |
|----------|--------|
| **Which country eats most beef/pork/poultry?** | Top Countries Table |
| **How has meat consumption changed since 1990?** | Line Chart (trends) |
| **Which meat grows fastest by country?** | Bar + Line charts |
| **Top meat in USA vs Brazil vs others?** | Country slicer + Bars |
| **Country vs world average?** | Gauge (27.26kg avg) |

**Interactive Features:**
- **Country dropdown** - USA, Brazil, Argentina, etc.
- **Meat slicer** - Beef/Pig/Poultry/Sheep  
- **Year slider** - 1990→2028
- **Live updates** - All charts filter instantly

**Key Insights:**
- **Argentina leads beef** (55kg/person)
- **Poultry rising fastest** globally
- **Sheep declining** everywhere
- **USA above world avg** (27kg/person)

**How to use:**
1. Download the '.pbix' file
2. Open in **Power BI Desktop** (free)
3. Explore with slicers!


**This OECD-FAO Agricultural Outlook dataset includes data beginning from 1990 through to projections for 2020-28.
The csv file contains columns: LOCATION, SUBJECT, MEASURE, TIME, and VALUE.**
**Data:** Per capita KG from OECD (1990-2028 forecast)
